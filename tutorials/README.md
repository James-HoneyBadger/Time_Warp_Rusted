## How to Use These Tutorials

1. **Open Time Warp Rusted** and select the language you want to learn.
2. **Follow each lesson in order** — they build on each other.
3. **Type the code yourself** rather than copying and pasting. Typing helps
   you learn syntax and spot errors.
4. **Experiment!** Change numbers, add lines, and see what happens.
5. **Check the Examples/ folder** for more programs to study.

## Difficulty Levels

- 🟢 **Beginner** — No prior experience needed (BASIC, Logo, PILOT)
- 🟡 **Intermediate** — Some programming concepts helpful (Pascal, Forth)
- 🔴 **Advanced** — Prior programming experience recommended (C, Prolog)
