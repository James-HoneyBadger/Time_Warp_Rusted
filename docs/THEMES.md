# Themes & Appearance Guide

Time Warp Rusted ships with **20 built-in themes** across three categories,
plus configurable font families and sizes. Customize your workspace from the
**Appearance** menu or the **Themes** panel in the left sidebar.

## Changing Themes

### From the Appearance Menu

1. Open the **Appearance** menu in the menu bar
2. Mouse over a theme category (🕹 Retro, 🌙 Dark, or ☀ Light)
3. Each theme shows a **color swatch** preview alongside its name
4. Click to apply immediately

### From the Themes Panel

1. Click **Themes** in the left sidebar panel
2. Browse themes organized by category
3. Click any theme name to apply it

### Keyboard Shortcut

There is no direct keyboard shortcut for theme switching, but you can
quickly access the Appearance menu with **Alt** key navigation.

## Fonts

### Font Families

Seven font families are available:

| Font | Style |
|------|-------|
| **Monospace** | System default monospace (default) |
| **Hack** | Open source font for code |
| **Fira Code** | Mozilla's programming font with ligatures |
| **JetBrains Mono** | JetBrains IDE font |
| **Source Code Pro** | Adobe's monospace font |
| **Consolas** | Microsoft's code-optimized font |
| **Courier New** | Classic typewriter monospace |

Change the font from:
- **Appearance → Font Family** submenu
- The **font dropdown** in the editor toolbar

> **Note**: Font availability depends on your operating system. If a selected
> font is not installed, the system will fall back to the default monospace
> font.

### Font Sizes

Six presets are available from the **Appearance → Font Size** menu:

| Preset | Size |
|--------|------|
| Tiny | 10pt |
| Small | 12pt |
| Medium | 14pt (default) |
| Large | 16pt |
| X-Large | 20pt |
| Huge | 24pt |

You can also adjust the font size from the editor toolbar, or use keyboard
shortcuts:

| Shortcut | Action |
|----------|--------|
| **Ctrl + =** | Increase font size |
| **Ctrl + −** | Decrease font size |

| Element | Description |
|---------|-------------|
| Background | Main window background |
| Panel background | Side panels, toolbars |
| Editor background | Code editing area |
| Editor text | Default code text |
| Line numbers | Line number gutter |
| Current line | Highlighted active line |
| Selection | Text selection highlight |
| Menu bar | Top menu background |
| Button | Standard button color |
| Button hovered | Button mouse-over state |
| Tab active / inactive | Tab bar colors |
| Output background | Output panel background |
| Output text | Output panel text |
| Error text | Error message color |
| Canvas background | Drawing canvas background |
| Keywords / Strings / Comments / Numbers | Syntax highlighting colors |

---

## Tips

- **Retro themes** pair well with **Courier New** or **Monospace** fonts
  for an authentic period look.
- **Dark themes** are recommended for long coding sessions to reduce eye
  strain.
- **Increase font size** to Huge (24pt) when presenting or teaching.
- Theme changes take effect **immediately** — no restart required.
- Your selected theme and font persist across sessions.
