# Keyboard Shortcuts Reference

Complete list of keyboard shortcuts in Time Warp Rusted.

---

## Program Execution

| Shortcut | Action |
|----------|--------|
| **F5** | Run program |
| **F6** | Stop program |
| **F7** | Step (execute one line) |

## File Operations

| Shortcut | Action |
|----------|--------|
| **Ctrl + N** | New file |
| **Ctrl + O** | Open file |
| **Ctrl + S** | Save file |
| **Ctrl + Shift + S** | Save As |

## Editor

| Shortcut | Action |
|----------|--------|
| **Ctrl + Z** | Undo |
| **Ctrl + Shift + Z** | Redo |
| **Ctrl + F** | Find |
| **Ctrl + H** | Find and Replace |
| **Ctrl + G** | Go to Line |

## View & Layout

| Shortcut | Action |
|----------|--------|
| **Ctrl + 1** | Switch to Editor tab |
| **Ctrl + 2** | Switch to Output tab |
| **Ctrl + 3** | Switch to Canvas tab |
| **Ctrl + L** | Toggle left panel |
| **Escape** | Close menus / cancel input |

## Font Size

| Shortcut | Action |
|----------|--------|
| **Ctrl + =** | Increase font size |
| **Ctrl + −** | Decrease font size |

## Language

| Shortcut | Action |
|----------|--------|
| **Ctrl + R** | Cycle to next language |

---

## Editor Toolbar

In addition to keyboard shortcuts, the editor toolbar provides buttons for:

- **Undo / Redo**
- **Cut / Copy / Paste**
- **Find / Go to Line**
- **Comment / Indent / Unindent**
- **Font family** dropdown
- **Font size** adjuster (+/−)
- **Toggle line numbers**
- **Toggle word wrap**
- **Tab size** selector

---

## Mouse Controls

### Canvas

| Action | Input |
|--------|-------|
| Pan | Click and drag |
| Zoom | Scroll wheel |

### Editor

| Action | Input |
|--------|-------|
| Select text | Click and drag |
| Place cursor | Click |
| Context menu | Right-click |

---

## Platform Notes

- On **macOS**, substitute **Cmd** for **Ctrl** in all shortcuts.
- Keyboard shortcuts work when the main application window has focus.
- Menu bar shortcuts are displayed next to each menu item.
